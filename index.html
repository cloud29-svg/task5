Part One: Whats Javascript?

Javascript is a programing language that enables interactive web pages. It is an essential part of web applications.

Four Uses of Javascript:

  1. Web Development:Javascript is used to create interactive websites and web applications.
  2. Server Apllications: With Node.js, Javascript is used for server-side programming.
  3. Mobile Applications:Frame works like React Native allow building mobile apps using Javascript.
  4. Game Developments:Javascript can be used to dvelop browser games and even more mobile games.
